## 🚀 **Getting Started**

### 1️⃣ **Clone the Repository**

```bash
git clone https://github.com/codernotme/login-temps.git
cd login-temps
```

### 2️⃣ **Navigate to a Template**

```bash
cd Nextjs/Login1
```

### 3️⃣ **Run & Customize**

Follow instructions in each template’s `README.md` for detailed setup.
